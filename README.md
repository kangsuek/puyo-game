# Puyo Puyo 게임

이 프로젝트는 Python과 Pygame을 사용하여 만든 Puyo Puyo 게임입니다.

## 설치 방법

1. 이 저장소를 클론합니다.
2. Poetry를 사용하여 의존성을 설치합니다:

   ```
   poetry install
   ```

## 실행 방법

게임을 실행하려면 다음 명령어를 사용하세요:

```
poetry run python -m puyo.main
```
